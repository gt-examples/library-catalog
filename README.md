# Library Catalog

A public library catalog demonstrating internationalization with General Translation.

**[Live Demo](https://library-catalog.generaltranslation.dev)** | **[General Translation Docs](https://generaltranslation.com/docs)**

## About

This example app showcases a library catalog interface with book listings, availability tracking, and genre filtering — all fully internationalized. It demonstrates how to translate both static UI elements and dynamic content using General Translation's gt-next library.

## GT Features Used

- `<T>` — JSX translation
- `<Num>` — Locale-aware number formatting
- `<Var>` — Variable interpolation in translations
- `<LocaleSelector>` — Language picker
- `getGT` — Server-side string translations
- `loadTranslations` — Local translation storage

## Getting Started

```bash
git clone https://github.com/gt-examples/library-catalog.git
cd library-catalog
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
